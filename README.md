# 🎓 Student Grade Management System (C++)

A simple console-based Student Grade Management System built using C++ and OOP concepts.

## Features
- Add student details
- Display all students
- Search student by Roll Number
- Automatic grade calculation

## Concepts Used
- Classes and Objects
- Vectors (Dynamic Storage)
- Functions
- Conditional Statements
- Menu Driven Program

## Grade Criteria
- 90+ : A
- 75-89 : B
- 60-74 : C
- 40-59 : D
- Below 40 : F

## How to Run

Compile:
g++ main.cpp -o student

Run:
./student
